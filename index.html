Oque é HTML?

Html é uma linguagem fundamental para criar páginas na web.

Na página inicial de um site, o HTML é essencial para definir a aparência e a navegação,

facilitando a interação do usuário e a apresentação de informações.

Combinado com CSS e JavaScript, ele forma a base de qualquer projeto web, tornando a criação de sites mais dinâmica e atraente.
